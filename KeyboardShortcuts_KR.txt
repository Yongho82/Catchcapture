키보드 단축키 안내

메인 창 단축키:
Ctrl+C - 선택한 이미지 복사
Ctrl+Shift+C - 모든 이미지 복사
Ctrl+S - 선택한 이미지 저장
Ctrl+Shift+S - 모든 이미지 저장
Delete - 선택한 이미지 삭제
Ctrl+Shift+Delete - 모든 이미지 삭제
Ctrl+A - 영역 캡처
Ctrl+F - 전체 화면 캡처
Ctrl+M - 간편 모드 토글
Ctrl+P - 선택한 이미지 미리보기 열기
Ctrl+Z - 실행 취소 (선택한 이미지를 원본으로 되돌림)
Ctrl+R - 스크롤 캡처
Ctrl+D - 지정 캡처 (픽셀 캡처)
Ctrl+O - 이미지 파일 열기
Ctrl+N - 새 캡처 (영역 캡처)
Ctrl+T - 항상 위 토글

미리보기 창 단축키:
Ctrl+C - 현재 이미지를 클립보드에 복사
Ctrl+Z - 마지막 편집 실행 취소
Ctrl+Y - 마지막 편집 다시 실행
Ctrl+S - 현재 이미지 저장
Ctrl+Shift++ - 확대
Ctrl+- - 축소
Ctrl+0 - 실제 크기로 재설정
Esc - 현재 편집 모드 취소

간편 모드 단축키:
Ctrl+D - 단축키 설정 창 열기
